通过github/hexo/nexT搭建个人博客

发表于 2018-05-13

文章整个搭建参考下面的文章，搭建时间大概一天左右。中间也踩了一点坑，不过 搜索一下 ，也是可以很快解决的。

唯一的一个大坑，大概就是，每次 hexo g 的时候总是不通过。 通过 强大的 搜索引擎。

发现在 _config.yml 文件中 # Site， #URL 的冒号 后面需要手动加空格，不然编译不过(哪怕那里没有内容，也要加空格)

至于其他的内容，参考下面的参考链接。都很详细 。

hexo的新建文章

  • hexo new 'xxx'
  • 找到资源文件_post下的md，更新文章
  • hexo clean
  • hexo s （查看本地）
  • hexo g (部署)
  • hexo d (发布)

其中hexo clean并不需要每次都使用
